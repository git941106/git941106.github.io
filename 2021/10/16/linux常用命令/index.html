常用命令

文件传输

上传文件

scp 要上传的文件路径 username@ip:目标路径

如果是文件夹，加参数scp -r 要上传的文件路径 username@ip:目标路径

下载文件

scp username@ip:服务器文件路径 下载到的文件路径

如果是文件夹，加参数scp -r username@ip:服务器文件路径 下载到的文件路径

查找文件

find -name filename 查找当前目录及其子目录下的文件

linux三剑客

grep

awk

sed

修改文件属性

chgrp groupname filename

排序

sort -k 1 -r -n
# -k指定列数 -r 反序，默认降序,-n 按照数值大小排序，默认按照ASCII字符大小排序 ，如果是字符串，只比较第一个# 字符 -t分割符，默认是空格

vi快捷键

vi有三个模式

  1. 命令行模式
  2. 末行模式
  3. 编辑模式

任何模式下按下esc都会回到命令行模式

vi 命令打开文件时，进入命令行模式

在命令行模式下，按下:会切换到末行模式

命令行模式下按下i进入插入模式(编辑模式)

命令行模式

gg 光标回到首行

shift+g 光标回到尾行

ngg 光标回到第一行

shift+4 光标回到行首

shift+6 光标回到行尾

nyy 复制n行

p 粘贴

末行模式

查看文件格式，windows，max，linux下，换行的格式是不一样的，windows是\r\n,max是\r,linux是\n

:set ff?

修改文件格式

:set ff=unix

替换

:%s/old/new/g %范围是整个文件s查找 old 查找的内容 new 替换成new g 全局

:2,15s/old/new/g 只替换2到15行的内容

查找

:/string 查找内容string，按下enter后，光标回到文件首行，n定位下一个找到的内容，N定位上一个找到的内容

格式化日期

date +%Y%m%d

解压缩

tar -zcvf traget.tar.gz 要压缩的目录或文件 # 压缩
tar -zxvf filename -C path# 解压到指定目录

shell

变量的引用

i=2
j=3
echo $i,${j},$[i*j],$((i*j))
# 变量引用可用$变量名，${变量名}
# 当要计算引用两个变量的算数值可用$[变量名+变量名],
#$((变量名+变量名))
array=(1 4  5 4)
array[3]=100 #赋值
echo ${array[0]} # 获取第一个值
echo ${array[-1]} # 获取最后一个值
echo ${array[@]} # 获取所有值
echo ${array[*]} # 获取所有值
echo ${#array[*]} # 数组长度
echo ${#array[@]} # 数组长度

获取命令执行结果

echo `date +%Y%m%d`
# 飘号表示可执行命令

let

执行表达式，不需要$引用变量

i=0
let i+=1 # 相当于i = $[i+1]
echo $i

统计当前 Linux 系统中可以登录计算机的账户有多少个

grep "bash$" /etc/passwd | wc -l
# $ 匹配结尾 wc -l 统计行数
#或者
awk -F':' '$7~/bash$/ {print $1}' /etc/passwd | wc -l
# -F 指定分隔符 $7~ 指定匹配的列数

获取执行时的参数

# bash args.sh 1 2
echo $0,$1,$2
#结果 args.sh,1,2
#./args.sh 1 2
#结果 ./args.sh,1,2

备份日志

# 备份日志 飘号里是命令，获取格式化的日期
tar -zcvf log-`date +%Y%m%d`.tar.gz /var/log
# 添加定时任务 crontab -l 查看定时任务
crontab -e
#在打开的文件中添加如下命令,表示每分钟执行一次
* * * * * /home/logback.sh

当前用户是否超级管理员

if [ $USER == 'root' ] # 判断当前用户是否是管理员$UID -eq 0
then
    echo '管理员'
else
    echo '非管理员'
fi

进度条

#!/bin/bash
echo -ne "\033[?25l" # 隐藏光标
for i in {1..100}
do
        echo -e "$i%\033[1A" #\033[1A 回到行首
        sleep 0.1
done

配置pip镜像加速

cd ~
mkdir .pip
cd .pip
echo [global]>pip.conf
echo index-url=https://mirrors.aliyun.com/pypi/simple/ >>pip.conf

配置docker镜像加速

mkdir /etc/docker
# 如果已经存在daemon.json文件，会覆盖文件内容
echo {"registry-mirrors": ["https://docker.mirrors.ustc.edu.cn"]} >daemon.json
